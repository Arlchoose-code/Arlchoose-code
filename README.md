## 👋 About Me

I'm **Syahril Haryono** — a linguist-turned-AI engineer from Bogor, Indonesia.

I studied **German Language Education** at Universitas Negeri Jakarta, but somewhere along the way I started wondering: *why doesn't Bahasa Indonesia have a proper open-source LLM built from scratch?*

That question became **Aibys** — a full LLM pre-training ecosystem for Bahasa Indonesia, built entirely from PyTorch, with a custom BPE tokenizer, a 13B-token corpus pipeline, and a Transformer architecture inspired by LLaMA and Mistral.

Beyond AI, I've spent years doing community work — including a digital transformation program in **Desa Medalsari, Karawang**, where I built and deployed a village website and trained locals to use it. That experience taught me something more valuable than any course: *technology fails when it ignores the humans it's built for.*

That's the intersection I work in — **language, AI, and communities that are still waiting to be included in the digital world.**

## 🔬 The Aibys Ecosystem

> A complete open-source pipeline for building an Indonesian Large Language Model from scratch.

| Repo | Description |
|------|-------------|
| 🗃️ [**Aibys-Data-Collector**](https://github.com/Arlchoose-code/Aibys-Data-Collector) | Download, clean, shuffle, and prepare Indonesian text datasets — RAM-efficient streaming for 50GB+ corpora |
| 🏗️ [**Indonesian-LLM-Starter**](https://github.com/Arlchoose-code/Indonesian-LLM-Starter) | Decoder-only Transformer from scratch: RMSNorm, RoPE, SwiGLU, Flash Attention 2, GGUF export |
| 🎯 [**Indonesian-LLM-Finetune**](https://github.com/Arlchoose-code/Indonesian-LLM-Finetune) | LoRA fine-tuning pipeline to turn a pre-trained Aibys into a conversational assistant |
| 🔤 [**aibys-tokenizer**](https://huggingface.co/syhrlhyn/aibys-tokenizer) *(HuggingFace)* | BPE tokenizer, 32K vocab, trained on 10M sentences with weighted sampling for Bahasa Indonesia |

**Status**: Pipeline complete. Proof-of-concept training run done (20K steps, coherent Indonesian text generation). Full training pending compute resources.

## 🌏 Why Indonesian NLP?

Bahasa Indonesia is spoken by **270 million people** — yet it remains severely underrepresented in the global NLP landscape. Most multilingual models treat it as an afterthought.

My work on Aibys is an attempt to change that, starting from the ground up: custom tokenizer, curated corpus (~13B token estimate), and a training pipeline that anyone with a GPU can run.

The broader goal? Build language technology that actually works *for* Indonesian communities — including rural ones that current tools completely ignore.
